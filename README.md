# Desafios com Docker e Microsserviços – Projeto 2

Este repositório reúne as implementações dos **5 desafios** do projeto de Docker e Microsserviços.

A ideia foi manter tudo o mais didático possível, com código simples e READMEs explicando em linguagem
humana o que está acontecendo em cada parte.

## Estrutura

- `desafio1/` – Containers em rede (server + client)
- `desafio2/` – Banco de dados com volume e persistência
- `desafio3/` – Orquestração com Docker Compose (web + db + cache)
- `desafio4/` – Dois microsserviços independentes se falando via HTTP
- `desafio5/` – API Gateway centralizando dois microsserviços

Cada pasta tem o seu próprio `README.md` com:
- objetivo do desafio;
- explicação da arquitetura;
- como subir os containers;
- como testar o funcionamento.
